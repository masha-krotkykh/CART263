Instructions

1. Hover the mouse over the image to activate it.
2. Use «Q» and «W» keys to change the size of the image.
3. Use «A» and «S» keys to rotate the image.
4. Use space bar to flip the image horizontally.
5. Drag the image to bring it to the front.
6. If there is an image you don't like, just throw it in the bin.
7. Press «I» to add a new image.
8. To resize the canvas drag right/bottom border.
